
# 3D Grid Earth Webpage

This is a code bundle for 3D Grid Earth Webpage. The original project is available at https://www.figma.com/design/zfut9aeEGv4gWRTWAjEjEo/3D-Grid-Earth-Webpage.

## Local setup

1. Use Node `20.19.0` or newer.
   `nvm use`
2. Install dependencies.
   `npm install`
3. Start the local development server.
   `npm run dev`

## Local verification

- Run the test suite.
  `npm test`
- Build the project once to confirm the environment is healthy.
  `npm run build`
- Preview the production build locally if needed.
  `npm run preview`
